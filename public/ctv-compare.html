Compare uploaded CSV vs live app-ads.txt

Choose CTV, Mobile, or Website, then upload a CSV and run compare. For CTV, the file should include App Bundle, app-ads.txt URL (or app_ads_txt_url), Inventory Partner Domain, and adline columns with YES / NO or Found / Missing. Fetches use pipeline priority (partner .txt link → partner host fallbacks → app-ads URL from the file). For Mobile, use the same columns except fetches use only the app-ads URL from the CSV (inventory partner is ignored). If that URL has no .txt path, the app loads /app-ads.txt on the same origin. Column order does not matter. For Website, include a Domain column (bare host or URL; http / https / www optional). The tool fetches ads.txt using http://{domain}/ads.txt, then http://www.{domain}/ads.txt, then HTTPS variants, in that order. Adline matching uses the same rules as Mobile. CSV has no cell colors — use Download Excel (colors) for green/red fills (CTV, Mobile, and Website).
